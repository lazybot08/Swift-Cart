Swift Cart API Endpoints

User Authentication Endpoints

Endpoint Name Endpoint path Description Authentication Authorization Request Method
Register User https://e-commerce-backend-y30k.onrender.com/api/v1/registerUser To register data in the database. No User Route Post Request Route
Login User https://e-commerce-backend-y30k.onrender.com/api/v1/loginUser If user is already registered in Database then to login into his account. No User Route Post Request Route
Update Password https://e-commerce-backend-y30k.onrender.com/api/v1/password/updatePassword The user is already logged in and want to update the password Yes User Route Put Request Route
Forgot Password https://e-commerce-backend-y30k.onrender.com/api/v1/password/forgotPassword The user has forgot the password No User Route Post Request Route
Logout User https://e-commerce-backend-y30k.onrender.com/api/v1/logoutUser To Logout the user No User Route Get Request Route

User Profile Endpoints

Endpoint Name Endpoint path Description Authentication Authorization Request Method
Get User Details https://e-commerce-backend-y30k.onrender.com/api/v1/profile To get the user Data once the user has logged in. Yes User Route Post Request Route
Upload User Avtar https://e-commerce-backend-y30k.onrender.com/api/v1/profile/uploadAvatar To upload the user avtar Yes User Route Post Request Route
Get Profile Image https://e-commerce-backend-y30k.onrender.com/api/v1/profile/getProfileImage To get the user Avtar Yes User Route Get Request Route
Update Profile https://e-commerce-backend-y30k.onrender.com/api/v1/profile/update The update the user profile details except avtar and password. For that we have different endpoints Yes User Route Put Request Route

User Admin Endpoints

Endpoint Name Endpoint path Description Authentication Authorization Request Method
Get All Users https://e-commerce-backend-y30k.onrender.com/api/v1/admin/users To get the data of all users Yes Admin Route Get Request Route
Get the details of a User https://e-commerce-backend-y30k.onrender.com/api/v1/admin/users/:id To get the details of a user with the help of his userId Yes Admin Route Get Request Route
Update User Role https://e-commerce-backend-y30k.onrender.com/api/v1/admin/users/update/:id To update the user role. There are two roles "user" and "admin". Used for authorization Yes Admin Route Put Request Route
Delete User https://e-commerce-backend-y30k.onrender.com/api/v1/admin/users/delete/:id To delete a user from the database Yes Admin Route Delete Request Route

Products Endpoints

Endpoint Name Endpoint path Description Authentication Authorization Request Method
Get All Products https://e-commerce-backend-y30k.onrender.com/api/v1/products To get all the products. It has query parameters like 'page' for pagination, 'category' for filter, 'keyword' for searching products. No User Route Get Request Route
Get Top Deals https://e-commerce-backend-y30k.onrender.com/api/v1/topDeals To get the top deals on the logic that all the products having discount greater than 15% are top deals. No User Route Get Request Route
Get Categories https://e-commerce-backend-y30k.onrender.com/api/v1/productCategories To get all the categories that are available in the databse. No User Route Get Request Route
Get Product Details https://e-commerce-backend-y30k.onrender.com/api/v1/product/:id To get the details of a product based on productId No User Route Get Request Route
Add Product Review https://e-commerce-backend-y30k.onrender.com/api/v1/product/reviews To add the review of a product Yes User Route Put Request Route
Get All Reviews https://e-commerce-backend-y30k.onrender.com/api/v1/product/reviews/:id To get all reviews of a product Yes User Route Get Request Route
Delete a Review https://e-commerce-backend-y30k.onrender.com/api/v1/product/reviews/delete To delete a review by the user. Only the user who reviewed the productt will be able to delete it. Yes User Route Delete Request Route
